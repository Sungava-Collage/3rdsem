TABLE ATTRIBUTES

This element include global attrbutes. The following attrbutes listed on this page are:

aline

This enumerated attrbute indicates how the table must be aligned inside the containing document.
It may have the following values:

  • Left: the table is disblayed on the left side of the document;
  • center: the table is disblayed on the center side of the document;
  • right: the table is disblayed on the right side of the document;

Using Note Do not use this attrbute, as it has been document <table> element should be styled Using CSS.
Set margin-left and margin-right to auto or margin to 0 auto be actieve an effect that is similar to the align attrbute.

bgcolor

This attrbute defines the background color of a table.It consists of a 6-digit hexadecimal code as defined in sRGB and is prefixed by '#'.
This attrbute may be used with one of sixteen predefined color strings:
